# Introduction to Input and Output Design

Har information system mein:
```text

```

```text
Toh output bhi galat aayega
```

# What is Input Design?

Input Design:
```text
Wo process hai jisme data ko system mein enter karne ka proper method design kiya jata hai
```

Input design ka main purpose hota hai:
- easy data entry
- accurate data entry
- less errors
- fast processing

# Importance of Input Design
