# UNIT 2 – Images and Graphics in Multimedia

# Introduction to Images in Multimedia

Images are one of the most important elements of multimedia. An image can explain information much faster than text because human beings understand visual information very quickly.

In multimedia systems, images are used to:
- Improve understanding
- Make presentations attractive
- Explain concepts visually
- Increase user interaction
- Create realistic experiences

Without images, multimedia applications would look dull and difficult to understand.

---

# Real Life Examples of Images in Multimedia

Images are used everywhere in modern digital systems.

Examples:
- Instagram photos
- Website banners
- YouTube thumbnails
- Video game graphics
- Online advertisements
- Educational diagrams
- Logos and posters

For example, a YouTube thumbnail with attractive graphics gains more clicks compared to plain text thumbnails.

---

# Importance of Graphics in Multimedia

Graphics play a major role in multimedia systems. Graphics help convert ideas into visual form so users can understand information easily.

---

# Why Graphics are Important

Graphics are important because they:
- Attract user attention
- Improve communication
- Increase understanding
- Make content professional
- Improve user experience
- Help in branding and marketing

---

# Example of Importance of Graphics

Suppose a teacher explains human heart only using text.

Students may find it difficult to understand.

But if teacher uses:
- Diagrams
- Colored illustrations
- Animations

students understand much faster.

# Applications of Graphics

Graphics are used in:
- Web designing

# Types of Graphics

Graphics are mainly divided into two categories:

1. Vector Graphics
2. Raster Graphics

---

# VECTOR GRAPHICS

Vector graphics are images created using mathematical formulas such as:
- Lines
- Curves
- Shapes
- Points

Instead of pixels, vector graphics use geometric objects.

---

# Features of Vector Graphics

- Resolution independent
- Scalable without quality loss
- Smaller file size
- Easy editing

---

# Real Life Example of Vector Graphics

Examples:
- Logos
- Icons
- Maps
- Typography
- Cartoons

---

# Example

When a company logo is enlarged on a billboard, quality remains sharp because vector graphics are used.

---

# Advantages of Vector Graphics

---

## 1. No Quality Loss

Vector graphics can be zoomed infinitely without becoming blurry.

---

## 2. Small File Size

They require less storage compared to high-quality raster images.

---

## 3. Easy Editing

Individual objects can be edited separately.

---

## 4. Best for Design Work

Perfect for logos and illustrations.

---

# Disadvantages of Vector Graphics

---

## 1. Not Suitable for Realistic Photos

Complex photographic details are difficult.

---

## 2. Requires Specialized Software

Needs graphic designing tools.

---

# Common Vector File Formats

| Format | Description |

---

# Software Used for Vector Graphics

Examples:
- CorelDRAW
- Adobe Illustrator
- Inkscape

---

# RASTER GRAPHICS

Raster graphics are images made using tiny dots called pixels.

Each pixel contains color information.

---

# What is a Pixel?

A pixel is the smallest unit of a digital image.

Thousands or millions of pixels combine to form an image.

---

# Real Life Example of Raster Graphics

Examples:
- Digital photographs
- Wallpapers
- Camera images
- Scanned documents

---

# Features of Raster Graphics

- Made using pixels
- Best for photographs
- Supports realistic images

---

# Advantages of Raster Graphics

---

## 1. Best for Detailed Images

Perfect for photographs and realistic pictures.

---

## 2. Rich Color Support

Supports millions of colors.

---

## 3. Realistic Appearance

Captures fine details and textures.

---

# Disadvantages of Raster Graphics

---

## 1. Quality Loss During Zoom

Images become blurry or pixelated when enlarged.

---

## 2. Large File Size

High-resolution images consume more storage.

---

# Common Raster File Formats

| Format | Description |
|---|---|
| JPG | Compressed image |
| PNG | Transparent image |
| BMP | Bitmap image |
| GIF | Animated image |
| TIF | High-quality image |

---

# Difference Between Vector and Raster Graphics

| Vector Graphics | Raster Graphics |
|---|---|
| Made using shapes | Made using pixels |
| No quality loss | Quality reduces on zoom |
| Smaller size | Larger size |
| Best for logos | Best for photos |
| Resolution independent | Resolution dependent |

# Attributes of Images

Images have different properties called attributes.

Main attributes are:
1. Image Size
2. Resolution
3. Color
4. Color Depth
5. Brightness
6. Contrast

---

# 1. Image Size

Image size refers to dimensions of image.

Example:
```text
1920 × 1080 pixels
```

---

# Importance of Image Size

- Large images give better quality
- Large images consume more storage

---

# 2. Resolution

Resolution means number of pixels present in image.

Higher resolution means:
- Better quality
- Sharper image

---

# Example

```text
72 DPI → Low quality
300 DPI → High quality
```

DPI = Dots Per Inch

---

# Real Life Example

Passport photo printing requires high DPI for clarity.

---

# 3. Color

Color makes images realistic and attractive.

Digital systems use:
- RGB model
- CMYK model

---

# RGB Color Model

Used in:
- Computers
- TVs
- Mobile screens

Colors:
- Red
- Green
- Blue

---

# CMYK Model

Used in printing.

Colors:
- Cyan
- Magenta
- Yellow
- Black

---

# 4. Color Depth

Color depth means number of bits used to represent colors.

Higher color depth means:
- More colors
- Better image quality

---

# Examples of Color Depth

| Color Depth | Colors Supported |
|---|---|
| 8-bit | 256 colors |
| 16-bit | 65,536 colors |
| 24-bit | 16 million colors |

---

# 5. Brightness

Brightness controls lightness and darkness.

---

# 6. Contrast

Contrast controls difference between light and dark areas.

---

# IMAGE FILE FORMATS

Different image formats are used for different purposes.

Each format has:
- Advantages
- Limitations
- Compression methods

---

# BMP FORMAT

BMP stands for Bitmap.

It is one of the oldest raster image formats.

---

# Features of BMP

- Uncompressed format
- High quality
- Large file size

---

# Advantages of BMP

- Excellent quality
- Simple format

---

# Disadvantages of BMP

- Very large file size
- Poor for internet usage

---

# Real Life Example

Windows Paint commonly uses BMP.

---

# PIC FORMAT

PIC is an older image format used in graphics applications.

---

# Features of PIC

- Supports compressed images
- Used in old graphics systems

---

# Limitations of PIC

- Less popular today
- Limited compatibility

---

# JPG / JPEG FORMAT

JPG stands for Joint Photographic Experts Group.

Most commonly used image format.

---

# Features of JPG

- Compressed format
- Small file size
- Good for photographs

---

# Advantages of JPG

- Saves storage
- Fast sharing
- Internet friendly

---

# Disadvantages of JPG

- Quality loss after compression
- Not ideal for transparency

---

# Real Life Example

Social media platforms use JPG heavily.

---

# PNG FORMAT

PNG stands for Portable Network Graphics.

---

# Features of PNG

- Lossless compression
- Transparency support
- Better quality than JPG

---

# Advantages of PNG

- Supports transparent backgrounds
- High-quality graphics

---

# Disadvantages of PNG

- Larger size than JPG

---

# Real Life Example

Logos and website graphics commonly use PNG.

---

# TIF / TIFF FORMAT

TIF stands for Tagged Image File Format.

---

# Features of TIFF

- Very high quality
- Professional image format
- Used in printing

---

# Advantages of TIFF

- Excellent quality
- Lossless format

---

# Disadvantages of TIFF

- Extremely large file size

---

# Real Life Example

Publishing industries use TIFF for magazine printing.

---

# Difference Between JPG, PNG and TIFF

| JPG | PNG | TIFF |
|---|---|---|
| Compressed | Lossless | High quality |
| Small size | Medium size | Large size |
| No transparency | Transparency support | Professional use |
| Best for photos | Best for graphics | Best for printing |

Images and graphics are extremely important in multimedia systems because they improve communication, interaction, and visual understanding. Vector graphics are best for scalable designs, while raster graphics are ideal for realistic photographs.

Modern multimedia applications heavily depend on image capturing devices, image formats, and image editing software. Tools like Photoshop and CorelDRAW help professionals create high-quality graphics for education, entertainment, business, advertising, and web development.
